# Orbit Store Listing Draft

## Short description

目標をAreaごとに整理し、今日やることと振り返りを続けやすくするゴール管理アプリ。

## Full description

Orbit は、目標を日々の行動に落とし込みやすくするためのゴール管理アプリです。

Routines、Projects、Resources の3つの視点で目標を整理し、今日見るべきこと、期限が近いこと、進捗中の項目をすばやく確認できます。

主な機能:

- Areaごとの目標管理
- Routines / Projects / Resources の分類
- 今日のタスク確認
- 期限が近い目標の確認
- 月次レビュー
- ローカル保存
- 任意のGoogle Drive同期
- JSONエクスポート/インポート

Orbit は無料で利用できます。入力した目標データは端末のブラウザストレージに保存され、Google Drive同期を有効にした場合のみ、ユーザー自身のGoogle Driveへバックアップされます。
